Eggs Minivan - Catalyst (Arrow Layout)-pok3r
=======================
NOTE: This started off as derivative of a port of jeebak's planck layout, for tv44, but added a bunch of the space commands from jetpacktuxedo, changed what didn't work from me, and added pok3r style media keys.

using the indicator lights as backlighting
